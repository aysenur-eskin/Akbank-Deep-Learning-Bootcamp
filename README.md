# Akbank-Deep-Learning-Bootcamp
Bu çalışma Akbank Derin Öğrenme Bootcamp'i için 3GB boyutundaki A Large Scale Fish veri seti üzerinde yapılan Kaggle çalışmasını çermektedir.
Bu çalışma, Akbank Makine Öğrenmi Bootcamp'i kapsamında derin öğrenme alanında başlangıç seviyede sınıflandırma, veri analizi, model geliştirme ve değerlendirme teknikleri konusunda deneyim kazanmak için .png formatındaki verilerden oluşan 3GB boyutundaki A Large Scale Fish veri seti üzerinde  Kaggle geliştirme ortamında yapılan çalışmayı içermektedir.

Kullanılan veri seti: https://www.kaggle.com/datasets/crowww/a-large-scale-fish-dataset/data
Kaggle notebook linki:  https://www.kaggle.com/code/ayenureskin/akbank-deep-learning-bootcamp
